# 📱 Mobile Recharge System

A simple Python project that simulates mobile recharge plans for different networks such as Jio, Airtel, and BSNL.

## 🚀 Features

- Multiple Networks Support
  - Jio
  - Airtel
  - BSNL

- Recharge Plans
- Data Booster Packs
- Coupon Codes
- Recharge Receipt
- Recharge Success/Failure Message

## 🛠 Technologies Used

- Python 3
- if, elif, else conditions
- User Input
- Boolean Variables

## 📦 Supported Coupon Codes

| Coupon Code | Discount |
|-------------|-----------|
| SAVE50 | ₹50 Off |
| SAVE100 | ₹100 Off |

## 📱 Supported Networks

### Jio
- Regular Recharge Plans
- Data Packs

### Airtel
- Regular Recharge Plans
- Data Packs

### BSNL
- Regular Recharge Plans
- Data Packs
